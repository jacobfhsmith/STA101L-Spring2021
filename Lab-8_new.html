ANOVA

Analysis of variance (ANOVA) has the following hypothesis test framework:

\(H_0:\) Numeric variable X has the same mean across all subgroups

\(H_A:\) At least one of the subgroups has a different mean than the others

aov1 = aov(HillaryTherm~Party, anes)

Simple Linear Regression

\[Y_i = \beta_0 + \beta_1 X_i + \epsilon_i, \text{ where } \epsilon_i \overset{iid}{\sim} N(0, \sigma^2) \text{ for all } i\]

Conditions:

  • Linearity: The relationship between X and the mean of Y is linear and residuals have zero mean.
  • Homoscedasticity: The variance of residuals is the same for any value of X.
  • Independence: Observations are independent of each other.
  • Normality: For any fixed value of X, Y is normally distributed.

lm()

lm() function has two required arguments: the data and the formula for the model. Y~X1 + X2 is the formula, with Y being the response and X1 and X2 being the predictors.

For example, lets fit a simple linear regression model for TrumpTherm (response) and GOPTherm (predictor):

lm_ex = lm(TrumpTherm~GOPTherm, data = anes)
summary(lm_ex)
